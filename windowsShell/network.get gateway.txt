Get Network Gateway
Preconditions: 
  - No need to open command prompt as an admin. 
  - Need to be connected to network you want the gateway for
  - If you are using WIFI, this is on your router
Main Success Scenario:
  1. Open command prompt
  2. type in "ipconfig"
  3. Press enter
  4. In results, look for adapter that is connected to internet.
        like "Wireless LAN adapter Wi-Fi"
  5. Look for "Default Gateway" under the adapter. 
  6. See IP address to the right of "Default Gateway" and the dots.
